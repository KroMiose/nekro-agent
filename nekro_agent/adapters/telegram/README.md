# Telegram 适配器

适合把 Agent 放进 Telegram 私聊或群组里，做国际化沟通、社区问答和多媒体内容处理。  
如果你的使用场景跨地区、跨语言，或者用户本来就在 Telegram 活跃，这会是非常自然的入口。

## 适合什么场景

- 社区问答：回答用户问题、分发资料、维持群内基本互动
- 国际化沟通：在跨地区群组或私聊中作为统一助手使用
- 多媒体处理：面对图片、文件、语音、视频这类内容时更方便
- 长期陪伴：作为持续在线的 Bot 提供聊天和任务协助

## 你可以用它做什么

- 在 Telegram 私聊和群组中持续提供对话能力
- 处理图片、文件、语音、视频等常见多媒体内容
- 支持 @提及，方便在多人聊天里精确触发机器人
- 适合社区运营、跨地域沟通和机器人陪伴场景

## 使用前你需要准备

- 一个已经创建好的 Telegram Bot
- Bot 已加入目标群组，或可以正常处理私聊消息
- 如果你的网络环境有要求，提前准备好可用代理

## 使用体验上有什么特点

- 私聊和群聊都比较适合长期在线的 Bot
- 多媒体内容支持比较丰富，适合做资料、素材、结果传递
- 用户习惯 @Bot 或直接私聊，学习成本通常比较低

## 使用时建议注意

- 群组里建议明确触发方式，避免无意义高频回复
- 处理大文件或多媒体时，建议先小范围测试传输体验
- 如果面向公开社区，建议提前设计好权限和回答边界
