# 企业微信 AI Bot 适配器

适合在企业微信里放一个面向团队成员的智能助手，用来承接问答、通知、资料发送和内部协同。  
如果你的目标是“让团队成员直接在企业微信里调用 Agent”，这个适配器通常是最顺手的入口。

## 适合什么场景

- 内部问答：回答团队常见问题、操作流程和知识库内容
- 通知播报：推送状态更新、日报汇总、任务提醒
- 办公协同：给团队成员补充资料、解释结果、辅助决策
- 统一入口：让成员在熟悉的企业微信里使用 Agent 能力

## 你可以用它做什么

- 在企业微信单聊和群聊里接收并回复消息
- 发送文字、图片和文件，满足常见办公沟通场景
- 让团队成员直接在熟悉的聊天入口里使用 Agent
- 适合内部问答、流程提醒、信息播报和轻量助手场景

## 使用前你需要准备

- 一个已开通并可用的企业微信机器人
- 机器人已具备进入目标聊天场景的权限
- 你希望交给 Agent 处理的典型问题、通知或协作内容

## 使用体验上有什么特点

- 团队成员无需切换工具，直接在企业微信里就能用
- 更适合内部信息和日常办公协作，不必额外引导用户学习新入口
- 文字、图片、文件这些常见办公内容都比较容易接入

## 使用时建议注意

- 先明确机器人在团队里的职责边界，避免“什么都能问”导致预期混乱
- 公共群里建议搭配明确触发方式，减少误触发
- 如果需要严肃通知，建议先在少量测试群验证展示效果
