# SSE 适配器

适合把自定义客户端、网页应用或轻量外部程序接进 Nekro Agent，让它们也能像聊天平台一样参与对话。  
如果你不是接入现成社交平台，而是想把自己的前端、小工具、内部系统或实验性客户端接进来，这类适配器会更灵活。

## 适合什么场景

- 自定义网页聊天界面
- 内部工具或业务系统的消息接入
- 轻量客户端、控制台工具、实验性终端
- 多个外部程序同时接入同一个 Agent 能力

## 你可以用它做什么

- 让多个外部客户端同时接收和发送消息
- 传递文字、图片、文件和用户信息，保持完整会话体验
- 根据需要控制消息确认、超时和文件传输范围
- 适合自定义应用接入、内部工具桥接和轻量会话终端场景

## 使用前你需要准备

- 一个能持续接收和处理消息的外部客户端或系统
- 明确你希望在客户端中呈现哪些消息能力
- 如果涉及文件传输或开放访问，提前规划好安全边界

## 使用体验上有什么特点

- 灵活度高，适合做“不是现成聊天平台”的接入
- 可以把 Agent 能力嵌入你自己的界面或业务流程里
- 比较适合需要自定义交互方式、显示方式和接入策略的场景

## 使用时建议注意

- 如果面向公网使用，建议配置访问控制
- 文件传输建议先限制范围，再逐步放开
- 先从单一客户端验证，再扩展到多客户端同时接入
