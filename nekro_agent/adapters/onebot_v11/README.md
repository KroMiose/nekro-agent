# OneBot V11 适配器

适合把 Agent 放进 QQ 群聊和私聊里，承担日常问答、群内辅助、通知提醒、资料分发和轻量运营这类工作。  
如果你的主要使用场景在 QQ，这通常会是最通用、也最容易上手的一类适配器。

## 适合什么场景

- 群助手：在群里回答问题、补充资料、维护群内秩序
- 私聊助手：作为个人陪聊、工具助手或通知入口
- 资料分发：把图片、文件、说明文字直接发到群或私聊
- 日常播报：定时提醒、状态通知、消息汇总

## 你可以用它做什么

- 在 QQ 群和私聊里稳定收发消息
- 发送图片、文件和 @提及，满足常见互动需求
- 让较长的回复以更清晰的形式展示，减少刷屏感
- 适合群助手、陪聊、通知播报和资料投递场景

## 使用前你需要准备

- 一个可正常工作的 QQ 机器人接入环境
- 机器人已经加入目标群聊，或能正常接收私聊消息
- 如果你要发文件，建议提前确认目标平台对文件大小和类型的限制

## 使用体验上有什么特点

- 更适合中文聊天环境和高频互动场景
- 群聊和私聊都很自然，适合把 Agent 当成“长期在线成员”
- 对于公告、长说明、操作指引这类内容，展示体验通常会比单纯一长段文本更友好

## 使用时建议注意

- 先确认机器人账号稳定，再逐步增加自动化强度
- 如果群成员较多，建议明确机器人触发方式，避免刷屏
- 做群内通知或资料投递时，建议先选少量群测试再扩展
