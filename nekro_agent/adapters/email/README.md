# Email 适配器

适合把邮箱收发纳入 Agent 的日常工作里，用来处理新邮件提醒、邮件整理、附件流转和基础回信。  
如果你的工作里经常要盯邮箱、分拣附件、整理来信内容，这个适配器会比普通聊天平台更偏“事务助手”。

## 适合什么场景

- 邮箱值守：持续接收新邮件并在系统内统一查看
- 附件处理中转：把邮件附件保存、整理、再次转发给其他聊天渠道
- 来信整理：总结近期邮件、快速定位指定邮件、提取关键信息
- 基础回信：用默认发件账户对外发送说明、通知或回复

## 你可以用它做什么

- 同时管理多个邮箱账户，统一查看和处理邮件
- 接收新邮件提醒，并把邮件内容带进聊天上下文里
- 自动保存附件，方便后续整理、转发和再次使用
- 适合邮箱值守、通知汇总、附件处理中转和邮件助手场景

## 使用前你需要准备

- 至少一个可用的邮箱账户
- 对应邮箱的 IMAP/SMTP 能力已经开通
- 推荐使用授权码或应用专用密码，而不是直接使用登录密码

## 使用体验上有什么特点

- 更像“邮件工作台”而不是普通聊天入口
- 一个邮箱账户会对应成一个独立会话，方便持续跟踪
- 新邮件、附件、邮件正文都能纳入 Agent 的上下文里继续处理

## 适合交给它处理的事情

- 帮你盯通知类邮箱或项目邮箱
- 快速整理近期邮件重点
- 找出带附件、带特定主题或来自特定发件人的邮件
- 把邮件内容转成更适合阅读和继续处理的形式

## 使用时建议注意

- 轮询间隔不要设得过短，避免被邮箱服务商限制
- 如果你配置了多个发信账户，建议明确默认发件人
- 涉及隐私或正式外发内容时，先确认发送对象和正文再发送
